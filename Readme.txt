RENT BOOK APP 
(PEMINJAMAN BUKU)

*About*

Program ini dapat memudahkan monitoring dalam peminjaman 
buku bagi user yang mendaftar. 

- User dapat melakukan penambahan buku
  untuk dipinjamkan dan dapat melakukan peminjaman
  buku dari user lain

- User dapat mengetahui buku yang dipinjam sudah dikembalikan
  atau belum


*Package*

- fmt
- time
- os
- os/exec


*Instalation*

1. Clone source code dari github https://github.com/FaturFawkes/Rent-Book.git
2. Jalankan rentbook.exe
3. Aplikasi akan mulai dan berjalan
4. Registrasi user jika belum memiliki akun
5. Login untuk dapat menggunakan semua fitur yang tersedia
